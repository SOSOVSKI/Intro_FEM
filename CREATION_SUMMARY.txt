2. CREATED 7 JUPYTER NOTEBOOKS (550+ KB total)
   - 01_strong_to_weak_form.ipynb (6.6 KB)
     Demonstrates the transformation from strong form PDE to weak form using 
     method of weighted residuals and integration by parts. Shows boundary 
     condition handling and weak form decomposition.

   - 02_galerkin_discretization.ipynb (6.9 KB)
     Covers Galerkin discretization - replacing weak form with finite-dimensional 
     expansion using shape functions. Derives element stiffness matrix and load 
     vector through coefficient matching.

   - 03_1d_element_stiffness.ipynb (5.7 KB)
     Demonstrates conversion of symbolic matrices to numerical kernels via 
     lambdify. Includes property verification and parameter studies on 
     stiffness scaling.

   - 04_exact_vs_gauss_integration.ipynb (6.2 KB)
     Explores numerical quadrature on reference elements - comparing exact 
     integration vs. Gauss rules for different polynomial degrees.

   - 05_reference_element_gallery.ipynb (7.3 KB)
     Showcases 2D and 3D reference elements (P1 tri, Q1 quad, Q2 quad, P1 tet, 
     P2 tet, Q1 hex). Includes visualization of shape functions and Kronecker 
     delta property verification.

   - 06_2d_triangle_poisson.ipynb (8.1 KB)
     Derives element matrices for 2D Poisson on P1 triangle. Covers gradient 
     pullback via Jacobian transformation and element tensor extraction.

   - 07_manual_assembly_2d.ipynb (7.6 KB)
     Demonstrates global assembly from element matrices. Shows element-by-element 
     loop, boundary condition application, and FEM solution visualization on 
     simple mesh.
