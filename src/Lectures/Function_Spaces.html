Function Spaces in Finite Element Methods

A Quick Intro to Hilbert, L², and Sobolev Spaces

Motivation: Why Special Function Spaces?

  • In FEM, we often transform strong forms of PDEs (like ∇⋅(k∇u)=f) into weak forms using integration by parts.

  • Weak forms look like: Find u such that \[ \int_{\Omega}(k\nabla u \cdot \nabla vd\Omega) = \int_{\Omega}fv d\Omega \] for all suitable “test functions” v.

  • This involves integrals of functions and their derivatives.

  • Question: What kind of functions u and v can we “legally” plug into these integrals? Do they need to be smooth? Can they have kinks?

Analogy: From Vectors to Functions

  • You know vector spaces like ℝ³: vectors, addition, scalar multiplication.

  • We measure vector “size” (norm) using the dot product: \[ ||x||^2 = {x_1^2 + x_2^2 + x_3^2} = x \cdot x \\ ||x|| = \sqrt{x_1^2 + x_2^2 + x_3^2} = \sqrt{x \cdot x} \\ \]

  • Functions can also form vector spaces! We can add functions (f+g) and scale them (c*f).

  • Challenge: How do we define a “dot product” and “size” (norm) for functions? How do we ensure our space is “complete” (no missing limits)?

Hilbert Spaces: The General Framework

  • A Hilbert Space is a vector space (often of functions) with an inner product \(\left < f,g \right>\).
  • The inner product lets us define:
    • Norm (Size): \(||f|| = \sqrt{\left < f,f \right>}\)
    • Orthogonality (Angle): f is orthogonal to g if \(\left < f,g \right>=0\).
  • Crucially, Hilbert spaces are complete: Every Cauchy sequence converges to a point within the space . This is vital for proving solution existence and convergence of approximations (like FEM!).

\(L^2(\Omega)\) Space: Square-Integrable Functions

  • The most common Hilbert space in introductory FEM.
  • Definition: \(L^2(\Omega)\) is the space of functions f defined on a domain Ω whose square has a finite integral: \[ \int_{\Omega} |f(x)|^2 dx < \infty \]
  • Think: Functions with finite “energy” or finite mean-square value.
  • Inner Product: \[ <f, g>_{L^2} = \int_{\Omega} f(x) g(x) dx \]
  • Norm (L² Norm): \[ ||f||_{L^2} = \sqrt{\int_{\Omega} |f(x)|^2 dx} \]
  • \(L^2(\Omega)\) is a Hilbert space. It’s also sometimes called \(H^0(\Omega)\).

L²(Ω) Examples

Code 
```{python}
#| code-fold: true
#| label: fig-l2-examples
#| fig-cap: 'Left: sin(x) is in L²[0, π]. Right: 1/x is NOT in L²[0, π] as its square diverges near x=0.'


import numpy as np
import matplotlib.pyplot as plt

# Domain
x1 = np.linspace(0.01, np.pi, 400) # Avoid x=0 for 1/x plot
x2 = np.linspace(0, np.pi, 400)

# Functions
f1 = np.sin(x2) # In L2[0, pi]
f2 = 1 / x1      # Not in L2[0, pi] because integral of 1/x^2 diverges at 0

# Plotting
fig, axs = plt.subplots(1, 2, figsize=(10, 4))

axs[0].plot(x2, f1, label='f(x) = sin(x)')
axs[0].fill_between(x2, f1**2, alpha=0.3, label='f(x)²')
axs[0].set_title('Function in L²[0, π]')
axs[0].set_xlabel('x')
axs[0].set_ylabel('f(x)')
axs[0].legend()
axs[0].grid(True)
axs[0].set_ylim(bottom=0)

axs[1].plot(x1, f2, label='g(x) = 1/x')
axs[1].fill_between(x1, f2**2, alpha=0.3, label='g(x)²')
axs[1].set_title('Function NOT in L²[0, π]')
axs[1].set_xlabel('x')
axs[1].set_ylabel('g(x)')
axs[1].legend()
axs[1].grid(True)
axs[1].set_ylim(0, 50) # Limit y-axis to show divergence

plt.tight_layout()
plt.show()
```

Note

  • Functions can be unbounded (1/√x near 0 on [0,1]) and still be in L².
  • Functions can be discontinuous and be in L².

Need for Derivatives: Sobolev Spaces

  • Our weak form: \[ \int_{\Omega}(k\nabla u \cdot \nabla vd\Omega) = \int_{\Omega}fv d\Omega \]

involves derivatives \(\nabla u \cdot \nabla v\).

  • Requiring functions \(u, v\) to be only in \(L^2(\Omega)\) is not enough!

  • We need their derivatives to also “behave well” (i.e., be square-integrable).

Problem

What if u is continuous but has a kink (like |x|)?

Its classical derivative isn’t defined everywhere or might not be in L².

Sobolev Space \(H^1(\Omega)\)

  • \(H^1(\Omega)\) is the simplest Sobolev space relevant to many FEM problems.

Definition

\(H^1(\Omega)\) is the space of functions \(f\) such that both \(f\) and its first weak derivatives are in \(L^2(\Omega)\). \[ H^1(\Omega) = \{ f \in L^2(\Omega) \mid \nabla f \in L^2(\Omega) \} \]

(Where \(\nabla f\) denotes the vector of first weak partial derivatives).

  • Weak Derivative: A generalization of the derivative using integration by parts. It allows functions with “corners” or “jumps” in their classical derivative (like \(|x|\)) to still have a derivative within this framework, provided it’s an \(L^2\) function.

  • \(H^1(\Omega)\) is also a Hilbert space.

\(H^1(\Omega)\) Inner Product and Norm

  • Inner Product: Includes terms for both the function and its derivative. \[ <f, g>_{H^1} = \int_{\Omega} f(x) g(x) dx + \int_{\Omega} \nabla f(x) \cdot \nabla g(x) dx \]

(Or sometimes weighted differently, but involves both parts).

  • Norm (\(H^1\) Norm): Measures both the size of the function and its derivative. \[ ||f||_{H^1}^2 = ||f||_{L^2}^2 + ||\nabla f||_{L^2}^2 \]

  • If \(||f||_{H^1}\) is finite, the function is “well-behaved enough” for typical second-order PDE weak forms (like heat equation, elasticity).

\(H^1(\Omega)\) vs \(L^2(\Omega)\)

Code 
```{python}
#| code-fold: true
#| label: fig-h1-examples
#| fig-cap: 'Left: x² is smooth, both function and derivative (2x) are in L²[-1, 1], so x² ∈ H¹[-1, 1]. Right: Step function is in L²[-1, 1], but its derivative (a Dirac delta) is not an L² function, so Step ∉ H¹[-1, 1].'


import numpy as np
import matplotlib.pyplot as plt

# Domain
x = np.linspace(-1, 1, 400)

# Functions
f1 = x**2  # In H1[-1, 1]
f1_deriv = 2*x # Derivative is also nice (in L2)

f2 = np.sign(x) # Step function. In L2[-1, 1]
# Derivative is undefined at 0 (or a Dirac delta distribution), not an L2 function.

# Plotting
fig, axs = plt.subplots(1, 2, figsize=(10, 4))

axs[0].plot(x, f1, label='f(x) = x²')
axs[0].plot(x, f1_deriv, label="f'(x) = 2x", linestyle='--')
axs[0].set_title('Function in H¹[-1, 1]')
axs[0].set_xlabel('x')
axs[0].set_ylabel('Value')
axs[0].legend()
axs[0].grid(True)

axs[1].plot(x, f2, label='g(x) = sign(x) (Step)')
# We don't plot the derivative easily, just note its nature
axs[1].set_title('Function in L²[-1, 1], but NOT H¹[-1, 1]')
axs[1].set_xlabel('x')
axs[1].set_ylabel('Value')
axs[1].legend()
axs[1].grid(True)

plt.tight_layout()
plt.show()
```
  • Smooth functions (like polynomials) are typically in \(H^1\).
  • Functions with “jumps” (like the step function) are often in \(L^2\)$ but not \(H^1\).
  • Functions with “kinks” (like |x|) can be in \(H^1\) because their weak derivative exists and is in \(L^2\). \[ d/dx |x| = sign(x), \quad \text{which is in} \quad L^2[-1,1] \]

Summary:

  • Weak Formulations: FEM relies on integral (weak) forms of PDEs.

  • \(L^2 (H^0)\): Guarantees that integrals of functions themselves (\(\int f v dx\)) make sense and are finite. Useful for source terms, mass matrices.

  • \(H^1\): Guarantees that integrals involving first derivatives (\(\int ֿ\nabla f \nabla v dx\)) also make sense and are finite. This space is often the natural “solution space” for many second-order problems (heat, elasticity, potential flow). Ensures stiffness matrices are well-defined.

  • Using these spaces provides a rigorous mathematical foundation for FEM, ensuring solutions exist and approximations converge.

Beyond \(H^1\)

  • Higher-order Sobolev spaces exist, like \(H^k(\Omega)\), which include functions with weak derivatives up to order k.
  • Specific spaces, such as \(H(\text{div}\)) and \(H(\text{curl})\), are useful for vector fields in fluid dynamics and electromagnetics.
